# STEP 1 
========

  git clone

#STEP 2
=======
go in folder of this project with terminal and write a command :

  npm install
  
#STEP 3
=======

create your DATABASE and MODIFY in config/bdd.js
*your UserName
*your Password
*your DATABASE Name

#STEP 4
=======

  npm run start

and go in your browser : localhost:8020
